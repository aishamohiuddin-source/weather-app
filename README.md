# 🌤️ Weather App

A simple weather app that shows real-time weather using OpenWeather API.

## 🚀 Features
- Search weather by city
- Show temperature, humidity, wind speed
- "Feels like" temperature
- Responsive design
- Error handling

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API

## ⚙️ Setup

1. Clone repo:
git clone https://github.com/YOUR-USERNAME/weather-app.git

2. Get API key:
https://openweathermap.org/api

3. Add in script.js:
const API_KEY = "YOUR_API_KEY";

4. Run:
Open index.html

## 📌 Usage
Enter city → Click Search → See weather
